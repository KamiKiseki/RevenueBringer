1) Open Railway project. Open EVERY service -> Settings -> Networking / Custom domains.
2) Find which service lists "autoyieldsystems.com". If it is NOT "hvac-engine", REMOVE the domain from that service.
3) On service "hvac-engine" ONLY -> add custom domain autoyieldsystems.com (and www if you use it). Save.
4) Redeploy hvac-engine.
5) Test: curl https://autoyieldsystems.com/health
   You MUST see JSON with "railway_service" and "railway_git_sha".
   If you see 404 or HTML, the domain is still not on hvac-engine.

Proof: curl https://autoyieldsystems.com/ (first lines must say FLASK API and show railway_git_sha).
If you see RevenueBringer, you are hitting a different Railway (or static) target.
